Semantik

Das Layout einer zukünftigen Webseite sollte sinnvoll sein, genau wie eine Zeitungs- oder Zeitschriftenseite. Die Seite verfügt über Abschnitte, Überschriften, Listen, Bilder, Textabsätze usw. Um all diese Fülle an Inhaltstypen zu beschreiben, gibt es entsprechende Tags. Das heißt, Semantik ist von Natur aus Teil von HTML, aber sie ist nutzlos, wenn sie falsch implementiert wird.

Semantisches Markup ist ein Ansatz zur Erstellung von HTML-Markup, der auf der Verwendung von Tags basiert, die ihrem Zweck gemäß der Spezifikation entsprechen und für die Art des auszuzeichnenden Inhalts geeignet sind.

Semantisches Markup bedeutet, dass Tags nicht auf der Grundlage ihrer Darstellung im Browser ausgewählt werden, sondern auf der Grundlage der Art und Struktur des mit Tags versehenen Inhalts.
