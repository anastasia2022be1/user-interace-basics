Erstelle eine html Datei mit dem typischen boilerplate HTML Code

Erstelle eine style.css Datei hinzu. Lade diese Datei in der HTML Datei.

Füge einen main tag hinzu. Der main Bereich soll einen lila Hintergrund kriegen, und 100vh hoch sein.

Füge einen footer hinzu. Dieser ist typischerweise ganz unten im body tag platziert.

Gebe dem footer folgende graue Hintergrundfarbe: #37393d Der Footer soll 400px hoch sein.

Füge eine h1, eine h2, und eine h3 in dem main tag hinzu. Gebe diesen Text. Unter jedem der Tags soll ein Paragraph mit Text sein.
